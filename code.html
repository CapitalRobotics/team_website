Our Autonomous System

Our autonomous system utilizes an April Tag path-based approach to navigate and make precise adjustments during the autonomous phase. By identifying April Tags, the robot follows a pre-planned path with accuracy and consistency, optimizing performance and achieving seamless integration with our hardware.

Autonomous Mode

Our April Tag path-based autonomous system ensures precise navigation and obstacle avoidance during the autonomous phase.

TeleOp Mode

With real-time control, our drivers use an intuitive interface to adapt to game conditions and make strategic moves.

Sensors and Analytics

Our robot uses an array of optical sensors to detect April Tags and navigate using a PIDF control system. This allows for precise real-time adjustments and optimal performance during the autonomous phase.
